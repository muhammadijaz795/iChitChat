# 💬 iChitChat - Real-time Text Chat Application

A lightweight and responsive real-time text chat application built with PHP, HTML, CSS, and JavaScript. iChitChat provides a seamless messaging experience with instant message delivery and a clean, user-friendly interface.

## ✨ Features

- **Real-time Messaging**: Instant message delivery and updates
- **User Authentication**: Register and login functionality
- **Private Messaging**: Direct messages between users
- **Message History**: View past conversations
- **Emoji Support**: Express yourself with emojis
- **Responsive Design**: Works on desktop, tablet, and mobile

## 🛠️ Built With

- **PHP** - Backend logic and database interactions
- **MySQL** - Database for storing users and messages
- **JavaScript** - Frontend interactivity and AJAX calls
- **HTML5** - Structure and content
- **CSS3** - Styling and animations
